# Online_Coding_Judge
Mini LeetCode Using Python+SQL(SSMS) (UI using Tkinter)

It is a Desktop-Based Mini Version of Platforms Like LeetCode, Codeforces where user read the problem, enter the solution for the problem and Run or Submit the Code to check whether the code is Suitable for the Problem Statement and Can pass the Testcases.

# Workflow

1. User Sees the Problem Statement
2. User Types the Code for the given problem
3. User Can Check for the sample test whether the sample testcase can be passed by using the Run Button
4. User Can then Submit the code
5. The result box will show whether the submitted code is correct or not and passed all testcases
6. User can also Check the History of their submitted codes

# Database Schema

- Provided in schema\database.sql (Used SSMS(SQL Server) For Database)

# How to Use

1. Clone the repository or download the files keeping the structure same.

2. Download the dependencies:
- pip install pyodbc

3. Create the Database Using the sql file(Using SSMS/ Drive: ODBC Driver 17/18)

4. Run:
- python gui.py
